My Goals

This project aims to create a new subgenre of games that merges the fighting and rhythm game genres. My intent for this project was also to challenge my skills in game development such that I could learn more about it. I aimed high from the start of the semester, but needed to rework my plan in terms of finding alternatives to the elements that I was adamant about having in the project, and discerning which elements would detract from the overall experience. My other intent for this project was to make it comprehensive with my IDM coursework, such that I would be implementing a number of technical skills and ideas that I have learned throughout the program. The project was programmed and compiled in GameMaker Studio 2, however the rhythm maps were created using a beat detection algorithm scripted with Processing3. Character sprites were animated by hand, and 3D elements were rendered with Autodesk Maya and stylized with Photoshop.
